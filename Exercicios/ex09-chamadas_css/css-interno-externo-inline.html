CSS inline:

Esse texto demonstra como funcionam os estilos CSS inline.
Nesse caso adicionamos o parametro/atributo style dentro da própria tag.

CSS interno:

Esse outro texto está sendo modificado por CSS interno dentro
do <head> do mesmo documento onde o HTLM esta escrito.

CSS externo:

Esse texto e todos os seus estilos estão sendo modificados por um documento CSS externo. o que
é uma grande vantagem quando se trabalha com vários documentos HTML que devem possuir um mesmo estilo.
